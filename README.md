# QuickMD

A macOS Quick Look extension that renders Markdown files when you press spacebar in Finder.

## Features

- Rendered Markdown preview with a toggle to view raw text
- GitHub and Basic themes with light, dark, and system color schemes
- Syntax highlighting for code blocks
- LaTeX math rendering (inline and display)
- GFM support: tables, task lists, strikethrough
- Local image embedding
- Adjustable font size (50%--150%)

## Install

1. Open **QuickMD.app**
2. If macOS shows "Apple could not verify this app": go to **System Settings > Privacy & Security**, scroll down, and click **Open Anyway**
3. In System Settings, go to **Privacy & Security > Extensions > Quick Look** and make sure **QuickMD** is toggled on

## Usage

Select any `.md` file in Finder and press **spacebar**. That's it.

Use the button in the top-right corner to toggle between rendered and raw views. Open the QuickMD app to change theme, color scheme, font size, and default action.
